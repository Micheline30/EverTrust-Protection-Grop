How to Onboard a New Agent

Step by step setup guide

Step 1 — Buy Twilio Number

Purchase a Phone Number

Log into your Twilio account → Phone Numbers → Buy a Number. Cost is ~$1/month. Assign it to the Evertrust Dialer TwiML app.

Step 2 — Create Agent Identity

Set Agent Identity String

Choose a unique identity string (e.g. jsmith_agent). This registers them in Twilio for inbound call routing.

Step 3 — Add Agent Here

Fill in the Form Above

Fill in the Add Agent form with their name, username, PIN, Twilio number, identity, and token server URL. Click Add Agent.

Step 4 — Share Credentials

Send Login Info

Send the agent their username and PIN. Direct them to the agency portal to log in. That's it — they're ready to dial.

Export Agent Data

Download agent list for your records

A CSV of all agent accounts (excluding PINs). Keep this backed up — agent data is stored in the database.
